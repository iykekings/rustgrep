## Simple grep tool with Rust

> Install

  Clone this repo, `cd rustgrep`

  ```sh
    cargo install --path .
  ```

> Run

 If you followed the install step above - just Run
 
  ```sh 
    rustgrep <cmd> <query>
  ```

  Example:
  ```sh
    rustgrep ls Cargo
  ```

  Or
  ```sh
    cargo run <cmd> <Cargo>
  ```

  > Test

  ```sh
    cargo test
  ```
